Local order help

How to use the order list

Local orders are the browser audit trail for this device.

  1. Start paymentThe sale is saved automatically when you start it, before the customer QR opens.
  2. LoadRestore an order into Sale Status before checking or marking it paid.
  3. Automatic detectionThe terminal watches for the exact new payment amount and advances to verification when it appears on-chain.
  4. Pick transactionIf needed, use the transaction picker in Manual verify to fill the transaction ID from recent blockchain activity.
  5. Confirm then mark paidConfirm the transaction, amount, address, and confirmation count before fulfillment.
  6. ReceiptsPaid orders include Share receipt and Print / Save actions without interrupting the current sale.
  7. ExportDownload the current page or every saved local order as CSV or JSON.

Delete removes a single unlocked local row. Clear local orders wipes the browser table.
