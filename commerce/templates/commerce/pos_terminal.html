Local order help

How to use the order list

Local orders are the browser audit trail for this device.

  1. Save orderAfter showing the QR, save the sale so amount, memo, quote, and status are preserved.
  2. LoadRestore an order into Sale Status before checking or marking it paid.
  3. Pick transactionUse the transaction picker in Sale Status to fill the transaction ID from recent blockchain activity.
  4. Confirm then mark paidConfirm the transaction, amount, address, and confirmation count before fulfillment.
  5. ExportDownload the current page or every saved local order as CSV or JSON.

Delete removes a single unlocked local row. Clear local orders wipes the browser table.
