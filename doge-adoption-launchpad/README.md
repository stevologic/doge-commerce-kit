# DOGE Adoption Launchpad

This package is built around one legal lever that can materially increase Dogecoin demand within six months: real-world usage.

The core idea is simple:

1. Make it easy for small merchants to accept DOGE.
2. Give customers a reason to spend DOGE this month, not "someday."
3. Turn each merchant into a local growth node with reusable assets and measurable goals.

## What is included

- `site/index.html`: launch page for the campaign
- `site/styles.css`: visual system and responsive layout
- `site/app.js`: merchant ROI calculator and copy actions
- `docs/strategy.md`: six-month strategy and execution logic
- `docs/merchant-playbook.md`: practical onboarding flow for merchants
- `docs/outreach.md`: copy for sponsor, merchant, creator, and community outreach

## Campaign concept

Run a public adoption program called `Spend DOGE Week`.

Each participating city or online niche gets:

- a merchant list
- a customer rewards mechanic
- creator amplification
- a proof-of-spend social loop
- a simple leaderboard

This is designed to create transactional demand for DOGE, not price manipulation.

## How to use

Open `site/index.html` in a browser to review or publish the landing page.

Use the docs to recruit:

- local businesses
- DOGE-friendly e-commerce stores
- community organizers
- meme creators
- influencers who can drive first-use behavior

## Why this approach

Price follows demand, liquidity, attention, and narrative. Legal, durable price support comes from adoption loops that create repeated buy/spend behavior. A coordinated merchant + consumer + creator program is a more credible six-month lever than passive branding alone.
