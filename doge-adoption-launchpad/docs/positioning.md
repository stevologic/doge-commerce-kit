# Positioning

## Core message

Dogecoin is easiest to grow when it is visibly useful.

The message is not:

- "buy because number go up"
- "hold because community"

The message is:

- "pay with DOGE here"
- "this business accepts DOGE today"
- "this community is using DOGE in the real world"

## Narrative ladder

1. DOGE is accepted.
2. DOGE is used.
3. DOGE is spreading.
4. DOGE is economically relevant.

This ladder matters because price narratives usually trail usage narratives when the proof is public and repeated.

## Messaging rules

- always show merchants, customers, and proof
- avoid financial promises
- emphasize participation over speculation
- repeat simple slogans until they become recognizable

## Suggested slogans

- Spend DOGE Here
- DOGE In Motion
- Real Merchants. Real Payments. DOGE.
- Use DOGE This Week
